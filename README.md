# MedSync AI — Snowflake PoC

Project scaffold for a MedSync AI demo using Snowflake.

Structure:
- `data/` — sample data
- `src/` — data generation and simulation scripts
- `snowflake/` — SQL setup and ETL scripts
- `streamlit_app/` — simple Streamlit UI
- `docs/` — architecture and data model notes

See `requirements.txt` for Python dependencies.
